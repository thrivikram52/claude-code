## Claude Code Plugins

Install any plugin with: `/plugin marketplace add <owner/repo>` or `/plugin install <name>`

### Productivity & Superpowers

| Plugin | Install | Description |
|--------|---------|-------------|
| **Superpowers** | `obra/superpowers` | Enhanced Claude Code capabilities — power-user toolkit |
| **Get Shit Done** | — | Similar to Superpowers, focused on getting things done | [github.com/gsd-build/get-shit-done](https://github.com/gsd-build/get-shit-done) |
| **Context7** | `upstash/context7` | Up-to-date documentation and code examples for any library via MCP |
| **Claude Mem** | `thedotmack/claude-mem` | Memory and context persistence across sessions |
